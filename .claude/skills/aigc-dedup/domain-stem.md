## 一、AIGC 检测器高频标记句式（实测验证）

以下句式来自 PaperYY / PaperPass 实测中标记为"高度疑似AI"的段落，优先级最高。

### 定义/介绍模板（检测器极度敏感）

| AI 模板 | 拆解方式 | 示例 |
|---|---|---|
| ...是一种基于...的...方法，能够... | 拆成两短句，去掉"基于" | "超滤分级是一种基于分子尺寸的膜分离方法，能够..."→"超滤分级利用分子尺寸差异来分离。具体做法是..." |
| ...是近年来发展起来的一种新兴的... | 删掉"近年来发展起来的"，直接说是什么 | "木质素胶体球是近年来发展起来的一种新兴的..."→"木质素胶体球是一类新型..." |
| 与传统的...相比，...具有...多重优势 | 拆成"和...进行比较，...在...方面更有优势"，然后分点说 | "与化学沉淀法相比，超滤分级具有操作温和...等优势"→"和化学沉淀法进行比较，超滤分级不需要有机溶剂，常温常压就能操作，也更容易放大到工业规模" |

### 因果关系模板

| AI 模板 | 拆解方式 | 示例 |
|---|---|---|
| 正是因为缺乏...，才让/导致... | 拆成"由于对...的研究还不够，..." | "正是因为缺乏对...的系统研究，才让...不明晰"→"由于对木质素结构-胶体球性能的关系还缺少整体研究，...还不清楚" |
| 这限制了...，也阻碍了... | 拆成两句，去掉"也" | "这限制了胶体球的可控制备，也阻碍了其产业化进程"→"这让胶体球的可控制备变得困难。产业化进程也因此受阻" |

### 递进/强调模板

| AI 模板 | 拆解方式 | 示例 |
|---|---|---|
| 特别是...而且...仍停留在... | 拆成三短句，每句独立 | "特别是分级对热解行为和胶体球形成性能的影响依然不明确，而且目前大多数研究仍停留在..."→"分级对热解行为和胶体球形成性能的影响，目前还不明确。大多数研究也只关注..." |
| 不仅...还...因此... | 拆成多句，每句一个意思 | 见"不仅...而且..."规则 |

### 方法论流水账模板

| AI 模板 | 拆解方式 | 示例 |
|---|---|---|
| 本研究以...为原料，采用...获得/制备... | "本文使用...当作原料，通过...得到了..." | "本研究以工业碱木质素为原料，采用超滤分级获得..."→"本文使用工业碱木质素当作原料，经过超滤分级得到了..." |
| 采用...对...进行... | 改用"...用...来..." | "采用SEM对样品进行表征"→"样品用SEM来观察" |
| 为...，采用...对...进行... | 先说目的，再用"具体做法是..."引出方法 | "为观察胶体球的微观形貌，采用SEM对样品进行表征"→"胶体球的微观形貌用SEM来观察。具体做法是..." |

### 数据描述模板（中高敏感度）

| AI 模板 | 拆解方式 | 示例 |
|---|---|---|
| 结果分析：经过...后，我们得到了... | 去掉"结果分析："和"我们"，直接说结果 | "结果分析：经过超滤分级后，我们得到了..."→"超滤分级以后，得到了..." |
| 这一趋势表明，...具有...，而...则可能因... | 拆成两个独立句 | "这一趋势表明中等分子量级分具有最低的羟基含量，而低分子量级分则因..."→"这说明中等分子量的级分羟基含量最低。低分子量级分因为..." |
| 分析结果表明，...呈单调递增趋势，从...逐步升高至... | 直接说变化，去掉"分析结果表明" | "S/G比值从L5级分的0.610逐步升高到L100级分的0.852，整体呈上升趋势" |
| ...呈现...的趋势/规律 | 改用"...表现出..."或直接描述 | "...呈现出先降低后升高的趋势"→"...先降低后升高" |

### 机理描述模板

| AI 模板 | 拆解方式 | 示例 |
|---|---|---|
| ...的形成遵循...机理 | 改成"...的形成过程可以分为...步"或"...是这样形成的：" | "胶体球的形成遵循'解聚—交联—稳定'三步机理"→"胶体球的形成分三步：解聚、交联、稳定" |
| 该机理与...提出的...一致，证实... | 拆成"这个看法和...的观点相同"，然后单独说证实了什么 |

### 致谢模板（检测器高度敏感）

| AI 模板 | 拆解方式 |
|---|---|
| 本论文是在...的悉心指导下完成的。从...到...，...都给予了... | 拆成 3-4 个短句，加入具体细节和个人感受 |
| 感谢...在...方面的指导和帮助，特别感谢...在...过程中给予的建议和支持 | 每句话只感谢一个人或一件事，不要用"在...方面"、"在...过程中"等套话 |
| ...是我学习的榜样 | 改成具体学到了什么 |
| 感谢我的家人，感谢你们多年来对我的理解、支持和无私付出 | 加入一个具体的小故事或细节，打破模板感 |

## 二、光谱/热分析数据描述模板（PaperPass 实测 AI 50-70%）

FTIR、TGA 等仪器分析段落的描述语言高度公式化，是检测器的重点目标。

| AI 模板 | 拆解方式 | 实测 AI 值 |
|---|---|---|
| 在X cm⁻¹处观察到归属于Y的Z伸缩振动峰 | 拆成："X cm⁻¹处有一个Z伸缩振动峰，它属于Y的贡献" | 58-70% |
| 以A为内标进行归一化后，B呈现C的趋势 | 去掉"进行归一化"，改成"以A为内标归一化后，B先降低后升高"（直接说趋势） | 65% |
| 四个级分的asym/sym比值在X至Y之间，符合Z的结构特征 | 拆成两句："四个级分的asym/sym比值落在X到Y这个范围。这个比值符合Z的结构特征" | 58% |
| C=O相对含量（A₁₆₈₂/A₁₅₁₂）在L5和L10中分别为...在L20中大幅降低至...而在L100中又回升至... | 拆成三句，每句说一个级分 | 58% |
| 这一梯度变化说明，...组分与...组分得到了有效分离，具有重要的工程实践意义 | 删掉"具有重要的工程实践意义"这种空洞评价 | 69% |
| 综上，不同分子量级分在...上存在系统性的结构差异 | 改成"从以上分析可以看出，不同分子量的级分在...上有明显的结构差异" | 57% |

### L3 补充规则 A：数据方向性描述的"公式化表达"深度拆解

以下"呈/呈现...趋势"句式是 AIGC 检测器的强信号，L3 必须拆掉双层包装：

| AI 公式化表达 | L3 深度拆解 | 说明 |
|---|---|---|
| 呈单调递增趋势 | 一直在增加 / 不断上升 | "呈...趋势"三个字本身就是检测器的强信号 |
| 呈先降低后升高的趋势 | 先降后升 / 先向下走再回升 | 去掉"呈现"+"趋势"双层包装 |
| 呈规律性变化 | 有规律地变化 / 变化是有规律的 | "规律性"→"有规律的" |
| 逐步升高至... | 慢慢升到... / 一路升到... | "逐步"→更口语化 |
| 随...增大呈...趋势 | 随着...变大，...也在不断... | 拆掉"呈...趋势"，让描述变成自然观察 |
| 从A的X逐步升高至B的Y | A是X，B是Y，中间呈上升态势 | 不用"从...至..."的AI压缩结构 |

### L3 补充规则 B：数据引述的去引导词化（删除"分析师口吻"）

L3 必须删除以下引导词和空洞评价，直接描述数据：

| V3 残留 / 常见表达 | L3 处理 | 优先级 |
|---|---|---|
| 这一梯度变化说明 | 直接说结论，不加任何引导词 | 最高 |
| 这一结果表明 / 上述结果表明 | 同上 | 最高 |
| 整体规律十分清晰 | 删掉整句（空洞评价） | 最高 |
| 从以上分析可以看出 | 删掉，或改成"可见" | 高 |
| 综上 | 删掉，或改成"总的来看" | 高 |
| 清晰呈现了 | 清楚地显示出 / 能清楚看到 | 高 |
| 分析结果表明 | 删掉，直接说结果 | 高 |
| 我们得到了... | 得到了...（去掉"我们"） | 高 |

### L3 补充规则 C：数字密集型句子的口语化拆分

当一个句子包含 3 个以上数据点时，AI 倾向压缩。人类写作会拆开：

| 压缩写法（AI） | L3 拆解（人类） |
|---|---|
| C=O相对含量在L5中是1.98，L10中是1.93，L20中大幅降到1.06，L100中又回升到1.98 | L5的C=O相对含量是1.98，L10是1.93。到了L20，这个值降到了1.06，降幅很大。而L100又回升到了1.98 |
| 四个级分的asym/sym比值在1.29到1.61这个范围，符合长烷基链的结构特征 | 四个级分的asym/sym比值落在1.29-1.61之间。这样的比值范围和长烷基链的结构特征是一致的 |

关键手法：逗号→句号，每级分一句，数据后面跟一句简短解释。

### L3 补充规则 D："存在+名词"压缩结构拆解

| 常见表达（AI） | L3 处理 |
|---|---|
| 在各特征峰的相对强度上存在显著差异 | 各特征峰的相对强度差别很明显 |
| 在官能团组成上有明显的结构差异 | 官能团组成差别很大 / 官能团组成不一样 |
| 存在少量粘连现象 | 有少量粘连 / 能看到一些粘连 |
| 存在系统性的结构差异 | 在结构上有系统性的差别 |

## 三、SEM 形貌观察描述模板（实测 AI 65-68%）

| AI 模板 | 拆解方式 | 实测 AI 值 |
|---|---|---|
| 所得产物未能呈现预期的X，而是表现为Y | 改成"产物没有出现预想的X，而是变成了Y" | 68% |

| 最让人惊喜的是，这些小球的表面非常干净 | 太口语化 → "这些胶体球的表面很干净，在高倍电镜下看不到毛刺或未成形的杂质" | 68% |

## 四、结论章节模板（实测 AI 56-70%）

结论章节有独特的 AI 模板，与正文不同：

| AI 模板 | 拆解方式 | 实测 AI 值 |
|---|---|---|
| 本研究以...为原料，通过...得到...，研究了...的影响。结论如下： | 拆成："本文用工业碱木质素做原料，先经过超滤分级得到不同分子量的组分，然后分析了分级对热解行为和胶体球形成的影响。主要结论有这些：" | 56% |
| 随着...降低，...明显下降，...也提前了很多 | 去掉"随着...降低"模板开头 | 56% |
| 尽管本研究取得了一定的成果，但仍存在以下不足之处： | 改成"本研究还有一些地方做得不够好：" | 63% |
| 未来，随着...的进一步完善和...的深入，有望建立更加精准的...模型 | 拆掉嵌套从句："以后实验数据更多、机理研究更深入以后，可以建立更准的分子量-胶体球性能关系模型" | 63-70% |
| 也希望本文能够为...贡献绵薄之力 | 太模板化 → 删除或改为具体的展望 | 63% |

### L3 补充规则 E：结论总结句的"评价口吻"消除

结论章节中常见以下 AI 评价口吻，L3 必须替换为更自然的人类表达：

| AI 评价口吻（V3 残留） | L3 处理 | 问题诊断 |
|---|---|---|
| 明显收窄了 | 收窄了很多 / 变得很窄 | "明显"+"了"组合是AI标签 |
| 大幅降低 | 降了很多 / 降幅很大（给出具体数字更好） | "大幅"是AI高频词 |
| 影响非常显著 | 影响很大 / 有很大的影响 | "非常显著"是双AI词叠加 |
| 开拓了新的思路 | 提供了一种新的方向 / 打开了新的研究角度 | "思路"+"开拓"双AI词 |
| 可以建立更加精准的...模型 | 有可能建立更准确的...关系模型 | "更加精准的"是AI偏好表达 |
| 初步说明了...之间的内在联系 | 初步弄清楚了...之间的关系 | "说明"+"内在联系"=双AI词 |
| 是一项重要的手段 | 是一种重要的方法 / 是很关键的一步 | "手段"过度抽象 |
| 能够显著改善 | 可以明显改善 / 对...有很大改进 | "显著"+"改善"双AI词 |
| 这一新式分离方法为... | 这种新的分离方法给... | "新式"不够自然 |

### L3 补充规则 F：研究不足条目的去模板化

"1. 2. 3. 4." 编号列表式的研究不足是 AI 高频模板，L3 处理：

- 每个条目开头不要用"对...的研究还不够深入"、"缺乏...的系统比较"等固定开头句式
- 每个不足后面跟一句"为什么没做到"的具体原因，增加真实感
- 视情况改为段落体而非编号列表，减少 AI 痕迹
- "未找到...的有效路径"→"目前还没有找到...的方法"
- "离实际应用还有差距"→"从实验室到工业化还有一段路要走"

## 六、致谢模板（第二轮加深，实测 AI 67%）

第一轮只改了开头句式，整段仍被标记。需要更深层拆散：

| AI 残存模板 | 进一步拆解 |
|---|---|
| 从论文的选题、实验方案的设计、研究过程的实施到论文的撰写与修改 | 拆成："选题的时候、设计实验方案的时候、做实验的时候、写论文改论文的时候" |
| 给予了无私的指导和宝贵的建议 | 改成"给了很多指导和建议" |
| 严谨的治学态度、渊博的专业知识、敏锐的科学洞察力和诲人不倦的师者风范 | 四个并列四字格太AI化 → 拆成两句，只留 1-2 个关键词 |
| 使我受益匪浅 | 改成"让我学到了很多" |
| 在此，谨向...表示最诚挚的感谢和最崇高的敬意 | 删掉"在此，谨向"，直接"衷心感谢..."删除"最崇高的敬意" |

## 七、参考文档对照例句库（76 组）

以下是从参考文档与检测报告中提取的对照例句。格式：原文（AI风格）→ 参考文档（人类风格）+ 手法说明。

### 摘要对照（10 组）

| # | 原文（AI 风格） | 参考文档（人类风格） | 手法 |
|---|---|---|---|
| 1 | 木质素**作为一种**...，**其**高值化利用...**意义重大** | 木质素**是一种**...，**对其进行**高值化利用...**具有重要意义** | "作为"→"是"、"其"拆解 |
| 2 | **可按**分子尺寸...**获得**结构**相对**均一的组分 | **可以依据**分子尺寸...**得到**结构**比较**均一的组分 | 可→可以、按→依据、获得→得到、相对→比较 |
| 3 | **然而**该分级...性能**及**热解动力学的影响机制**尚不明确** | **不过**该分级...性能、热解动力学的影响方式**还不清楚** | 然而→不过、及→、、尚→还、机制→方式 |
| 4 | **本研究以**工业碱木质素**为**原料 | **本文使用**工业碱木质素**当作**原料 | 以...为→使用...当作 |
| 5 | **系统考察**其热解行为 | **整体分析**其热解行为 | 系统→整体 |
| 6 | 多分散指数**显著降低** | 多分散指数**有很大下降** | 显著→有很大 |
| 7 | 分子量是控制形貌和尺寸的**关键因素** | 分子量**决定了**胶体球形貌和尺寸，**是一个重要因素** | 拆"是...的关键因素" |
| 8 | **揭示了**...之间的内在关联 | **说明了**...之间的内在关联 | 揭示→说明 |
| 9 | 提出**最佳策略为** | 提出...**的方法** | 策略→方法 |
| 10 | 提供了理论依据**和新思路** | 提供了理论依据与**新的技术方向** | 思路→技术方向 |
